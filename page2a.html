ScoutTrek Automated Posting

Social media only works if you show up consistently. For a Scouts organization, that's one more thing nobody has time for.

I built a pipeline that takes the whole process off their plate. Photos come in, the system analyzes them, writes the posts, checks for compliance, and publishes — without anyone having to think about it.

Workshop & Wander

I wanted to find out how much of the content creation process I could actually remove.

So I built a test — one AI-driven character publishing short educational videos across ancient cultures, unusual animals, science, and history. The system picks topics, checks what's already been posted to avoid repetition, writes the scripts, and generates descriptions and hashtags automatically.

The piece I couldn't fully close was video generation itself. The scripts and prompts were ready to hand off to a video tool — but connecting it end to end would have added costs I wasn't ready to take on for a test. So that step stayed manual.

26,000 views and nearly 300 engagements in a month. The system did what I needed it to do.

Project · Build Log

Chef's Hat

I love meal prepping but I wanted to do it on a budget without spending an hour planning every week.

I built a Zapier workflow where I send one Slack message — just the cuisine I want — and it comes back with a full recipe, serving sizes, and a grocery list with clickable links.

This was one of my first automation projects and it still sticks with me. I keep thinking about what I could build if I came back to it now.
